Probabilistic Timed Automata¶

Python library for building and simulating probabilistic timed automata.

Installation¶

To install using pip just use:

$ pip install probabilistic-timed-automata

For developers, since this project uses poetry for building/dependency management, please install that (and familiarize yourself with it). Then, you can run:

$ poetry install
